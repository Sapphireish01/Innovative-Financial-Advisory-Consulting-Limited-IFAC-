Innovative Financial Advisory (IFAC) SPA

This is a Single Page Application (SPA) for Innovative Financial Advisory (IFAC), a Lagos-based platform providing investment guidance, strategic consulting, and financial advisory services. The SPA is designed for a smooth, responsive, and modern user experience.

Features
About, Vision, Services, Stories, Contact

Responsive Design: Optimized for both desktop and mobile users.

Tech Stack

Frontend: React.js, Tailwind CSS


UI Components: React Icons
